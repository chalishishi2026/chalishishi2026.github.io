# 肆顾门数字文化公司门户网站

## 项目简介

肆顾门数字文化（南通）有限公司官方网站，主打有声行业相关业务。

## 功能模块

✅ **首页 (index.html)**
- Banner轮播图
- 核心业务展示（配音制作、广播剧创作、有声书录制、配乐制作）
- 最新资讯预览
- 精选作品展示
- 团队风采预览

✅ **行业资讯 (news.html)**
- 资讯列表页
- 分类筛选（行业动态、技术前沿、公司新闻）
- 分页功能
- 响应式设计

✅ **团队风采 (team.html)**
- 团队成员卡片展示
- 分类筛选（配音演员、编剧、制作人）
- 声音Demo播放功能
- 作品链接

✅ **原创剧本馆 (works.html)**
- 剧本卡片展示
- 分类筛选（广播剧、有声书、短剧）
- 封面图展示
- 在线试听功能
- 分页功能

✅ **创作者中心 (submit.html)**
- 用户注册/登录
- 作品上传表单
- 封面图、剧本文本、配图、BGM上传
- 上传进度显示
- 我的作品管理

## 技术栈

- 前端：原生 HTML5 + CSS3 + JavaScript（无框架依赖）
- 样式：CSS变量 + Flexbox + Grid + 响应式设计
- 部署：GitHub Pages + GitHub Actions 自动部署
- 后端：CloudBase（腾讯云开发）— 待集成
- 数据库：CloudBase数据库 — 待集成

## 使用说明

### 本地预览
直接用浏览器打开 `index.html` 即可预览。

### 配置GitHub Pages
1. 将代码推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择 `main` 分支和 `/ (root)` 目录
4. 等待部署完成（约1-2分钟）

### 自定义域名（待配置）
1. 在域名服务商添加CNAME记录，指向 `你的用户名.github.io`
2. 在GitHub仓库设置中填写自定义域名
3. 等待DNS生效（约10分钟-24小时）
